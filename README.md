Carelyst is a frontend healthcare web application developed during **PixxelHack Webathon 1.0**, organized by **TCET ACM SIG-AI**.

The project aims to provide quick and easy access to essential healthcare resources through an intuitive and user-friendly interface.

## 🚀 Features

- 🏥 Find nearby doctors
- 💊 Search for medications
- 📋 View basic prescriptions for common illnesses
- ❤️ Clean and responsive user interface
- 📱 Easy navigation with a modern design
  
## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 🎯 Project Objective

Healthcare assistance should be accessible and convenient. Carelyst was designed to help users quickly locate nearby healthcare services and access basic medical information during emergencies or common health situations.
